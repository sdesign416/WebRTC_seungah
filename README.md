# WebRTC-users

### 개요
선생님과 학생이 1:1 화상 채팅방을 통해 실시간으로 소통 <br>
간단한 문제풀이, 학습 지도 지원 <br>
WebRTC를 활용한 실시간 영상·음성 통신 구현 <br>

### 주요 기능
- 회원가입 및 로그인
- 선생님: 화상 수업방 생성
- 학생: 방 목록 확인 및 입장
- WebRTC기반 1:1 영상·음성 통신
- 마이크 및 스피커 설정
- 카메라 온오프, 전면/후면 전환

### 회원가입 및 로그인 기능
1. 아이디 중복 확인 절차 분리
	- 수업: 회원가입 버튼 클릭 시 아이디 중복여부 함꼐 확인
	- 개선: 아이디 입력단계에서 중복확인버튼 클릭하여 검증절차 분리
	- 중복 확인이 완료되지 않으면 회원가입 진행이 되지 않도록 처리
2. 사용자 역할에 따라 서비스(Lobby)화면 분리
	- radio 버튼을 통해 선생님과 학생 역할을 선택할 수 있도록 구현
	- 선택된 역할에 따라 로그인 후 다른 Lobby화면으로 이동
		: 선생님 Lobby: 화상 수업방 생성
		: 학생 Lobby: 생성된 화상 수업방 목록 확인 및 입장

### 메인 Lobby        
1. 사용자 타입에 따른 Lobby 구조 분리
	- 수업: 여러 채팅방을 사용자가 직접 선택하고 이동
	- 개선: 회원가입 시 저장된 userType에 따라 서로 다른 Lobby화면, 기능 제공
2. 선생님용 Lobby
	- 문제 정보 및 난이도를 설정하여 화상 채팅방 생성
	- 학생 입장 후 1:1 화상 수업 진행
3. 학생용 Lobby
	- 생성된 화상 채팅방 목록 및 정보 확인
	- 원하는 방 선택하여 1:1 화상 학습 참여
4. 실시간 소통 기능
	- 현재 접속자 목록 확인
	- 사용자간 실시간 채팅 기능 제공
